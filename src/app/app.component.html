Pipes Angular

Los pipes son herramientas de angular que nos permiten cambiar de forma visual la informacion. como lo vemos en la tabla, podemos tener strings y modificar su aspecto visual, pero el valor no cambia. es muy util para dar formatos de fecha, tipos de monedas, redondeos numericos etc. Aqui mostrare ejemplos de su uso

Variable Pipe Salida
{{ nombre2 }} capitalizado/personalizado {{ nombre2 | capitalizado }}
{{ nombre }} uppercase {{ nombre | uppercase }}
{{ nombre }} lowercase {{ nombre | lowercase }}
{{ nombre }} slice:3 {{ nombre | slice:3 }}
{{ nombre }} slice:0:3 {{ nombre | slice:0:3 }}
{{ arreglo }} slice:1:5 {{ arreglo | slice:1:5 }}
{{ PI }} number {{ PI | number }}
{{ PI }} number sin decimales {{ PI | number:'.0-0' }}
{{ PI }} number con 2 decimales {{ PI | number:'.2-2' }}
{{ porcentaje }} percent {{ porcentaje | percent }}
